# 数据库脚本说明

## 文件说明

### 1. init_database.sql
主数据库初始化脚本，包含完整的测试数据：
- 创建root和admin账户
- 插入测试产品数据
- 插入测试客户数据  
- 插入测试营销计划
- 插入测试交易记录
- 设置角色关联

**使用方式**：首次部署时在phpMyAdmin中执行

### 2. init_roles.sql
仅更新角色和账户ID的脚本：
- 安全地更新现有账户的ID和角色
- 处理主键冲突问题
- 重新分配角色关联

**使用方式**：当需要单独更新角色系统时执行

### 3. generate_password.go
生成bcrypt密码哈希的工具：
```bash
go run generate_password.go your_password
```

### 4. setup_data.go
程序化初始化数据的Go脚本（可选）

## 账户信息

| 用户名 | 密码 | 角色 | ID | 权限说明 |
|--------|------|------|-----|----------|
| root | root123 | 超级管理员(role=1) | 1 | 拥有全部权限 |
| admin | admin123 | 管理员(role=2) | 2 | 无系统管理和授权码权限 |

## 角色值说明
- `role=1`: 超级管理员
- `role=2`: 管理员
- `role=3`: 普通用户

注意：角色值越小，权限越高
